WebSocket Endpoint

Connect to the WebSocket server to send and receive messages:

ws://35.239.115.20:8000/ws/items/

Example Usage


    // JavaScript example to connect to the WebSocket
    const socket = new WebSocket("ws://35.239.115.20:8000/ws/items/");
    socket.onmessage = function(event) {
        console.log("Received:", event.data);
    };
